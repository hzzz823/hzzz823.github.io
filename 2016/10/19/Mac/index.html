快捷键和一些使用中的生僻问题

mac快捷键

  1. Command + ｀ 同应用不同窗口切换
  2. docker stop $(docker ps -aq) docker，停止所有
  3. docker rm $(docker ps -aq) docker，删除所有
  4. mvn install -Dmaven.test.skip=true maven跳过

Mac使用中的问题

中文输入法没有选字框

打开活动监视器，搜索输入法，找到中文输入法，关掉就好了。

Alfred搜狗输入法shift切换问题

  1. Features -> File Search -> Advanced -> Quick Look 勾去掉
  2. Advanced -> Force Keyboard 选搜狗

    App store使用已购项目再试一次

    shadowsocket调全局模式可以解决
