IntelliJ IDEA快捷键

Posted on 2016-10-17 | In 程序员

IntelliJ IDEA快捷键

  • Alt + Enter 补全
  • Ctrl + J 生成代码
  • Command + Alt + T 包围代码
  • Ctrl + O 覆盖方法
  • Ctrl + I 实现方法
  • Command + N 生成代码
  • Command + Alt + L 格式化代码
  • Ctrl + Alt + O 优化import
  • Command + D 复制当前行
  • Shift + Enter 开始新一行
  • Command + +,Command + -折叠/展开代码块
  • Double Shift 查询任何东西
  • Command + Shift + F 全局查找
  • Command + Shift + R 全局替换
  • Command + R 文件替换
  • Command + G 向下查找
  • Command + Shift + G 向上查找
  • Command + O 查找类文件
  • Command + Shift + O 查找所有类型文件、打开文件、打开目录，打开目录需要在输入的内容前面或后面加一个反斜杠/
  • Command + Alt +O 前往指定的变量/方法
  • Command + Shift + Delete 跳转到最后一个编辑的地方
  • Shift + F6 重命名文件
  • Command + Alt + M 将选中的代码提取为方法
  • Command + Alt + V 提取变量
  • Command + Alt + F 提取字段
  • Command + Alt + C 提取常量
  • Command + Alt + P 提取参数
  • Control + H 继承关系
